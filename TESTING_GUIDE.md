```
GET  http://localhost:8080/api/products/
GET  http://localhost:8080/api/products/1/
GET  http://localhost:8080/api/products/?category=dresses
GET  http://localhost:8080/api/products/?search=платье
```

### Категории
```
GET  http://localhost:8080/api/categories/
```

### Авторизация
```
POST http://localhost:8080/api/auth/login/
POST http://localhost:8080/api/auth/register/
POST http://localhost:8080/api/auth/logout/
GET  http://localhost:8080/api/auth/user/
```

### Заказы (требуется авторизация)
```
GET  http://localhost:8080/api/orders/
POST http://localhost:8080/api/orders/
```

### Отзывы
```
GET  http://localhost:8080/api/reviews/
POST http://localhost:8080/api/reviews/
GET  http://localhost:8080/api/products/1/reviews/
```

### Профиль (требуется авторизация)
```
GET  http://localhost:8080/api/profile/

```

## 🧪 Ручное тестирование

### 1. Тест регистрации
1. Откройте http://localhost:8080/auth/
2. Перейдите на вкладку "Регистрация"
3. Заполните форму:
   - Имя: Тест
   - Email: test@example.com
   - Телефон: +7 999 999-99-99
   - Пароль: test123
   - Подтверждение: test123
4. Нажмите "Зарегистрироваться"
5. Проверьте, что вы вошли в систему

### 2. Тест авторизации
1. Откройте http://localhost:8080/auth/
2. Введите:
   - Username: admin
   - Password: admin123
3. Нажмите "Войти"
4. Проверьте, что вы вошли в систему

### 3. Тест просмотра товаров
1. Откройте http://localhost:8080/catalog/
2. Проверьте, что отображаются товары
3. Попробуйте фильтры категорий
4. Попробуйте поиск

### 4. Тест добавления в корзину
1. На странице каталога нажмите "В корзину" на любом товаре
2. Проверьте уведомление
3. Откройте http://localhost:8080/cart/
4. Проверьте, что товар в корзине

### 5. Тест создания заказа
1. Добавьте товары в корзину
2. Откройте http://localhost:8080/cart/
3. Нажмите "Оформить заказ"
4. Заполните форму доставки
5. Подтвердите заказ
6. Проверьте, что заказ создан

### 6. Тест создания отзыва
1. Войдите в систему
2. Откройте товар
3. Оставьте отзыв с рейтингом
4. Проверьте, что отзыв отображается

### 7. Тест профиля
1. Войдите в систему
2. Откройте http://localhost:8080/profile/
3. Проверьте отображение данных
4. Измените телефон или адрес
5. Сохраните изменения
